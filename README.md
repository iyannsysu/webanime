# 🎬 IndoAnime - Website Streaming Anime Sub Indo

Website streaming anime subtitle Indonesia yang menggunakan API dari **Sonzai X Anime Lover V3**.

## ✨ Fitur

- 🏠 **Beranda** - Update anime terbaru
- 📺 **Anime Ongoing** - Daftar anime yang sedang tayang
- 📅 **Jadwal Rilis** - Jadwal tayang anime per hari
- 📋 **Daftar Anime** - List lengkap semua anime
- 🔍 **Pencarian** - Cari anime favorit
- 📖 **Detail Anime** - Informasi lengkap anime + daftar episode
- ▶️ **Streaming** - Nonton anime langsung

## 🛠️ Teknologi

- HTML5
- CSS3 (Responsive Design)
- JavaScript (Vanilla)
- API: Sonzai X Anime Lover V3

## 📁 Struktur File

```
anime-website/
├── index.html      # Halaman utama
├── style.css       # Styling website
├── app.js          # Logika aplikasi
└── README.md       # Dokumentasi
```

## 🚀 Cara Menggunakan

1. Buka file `index.html` di browser
2. Atau jalankan dengan local server:
   ```bash
   # Python
   python -m http.server 8000
   
   # Node.js
   npx serve
   ```
3. Buka `http://localhost:8000`

## 📡 API Endpoint

| Endpoint | Deskripsi |
|----------|-----------|
| `/anime/home` | Update terbaru |
| `/anime/ongoing` | Anime ongoing |
| `/anime/jadwal` | Jadwal rilis |
| `/anime/anime-list` | Daftar anime |
| `/anime/search` | Pencarian |
| `/anime/detail` | Detail anime |
| `/anime/stream` | Streaming video |
